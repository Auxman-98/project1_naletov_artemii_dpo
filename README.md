# Проект №1. Текстовая приключенческая игра "Лабиринт сокровищ"

## Обзор
Этот проект представляет собой консольную adventure игру, которая легко запустится на вашем компьютере с помощью терминала.

Представьте: вы - отважный исследователь в древнем лабиринте, полном хитрых коридоров, скрытых сокровищ и загадок, которые проверят вашу смекалку. В этой игре вы будете блуждать по коридорам, собирать артефакты в инвентарь, разгадывать ребусы и охотиться за главным призом.

Игра написана на чистом Python, без лишних зависимостей.

## Рекомендации по установке и запуску
Игра функциональна на операционных системах GNU/Linux с версией ядра не ниже 6.8. Также будет не лишним владеть базовыми навыками работы с терминалом в Linux.

- Проверьте, что на вашей системе установлен интерпретатор Python с версией не ниже 3.12.10
  - Если версия вашего интерпретатора ниже, установите нужную версию в системе по инструкции в материале №3 раздела [Вспомогательные ссылки](#vspomogatelnye-ssylki). Не забудьте перейти на установленную версию!
- Скачайте архив .tar.gz и извлеките его содержимое в удобную вам папку
- Через терминал установите библиотеку poetry с помощью команды `python3 -m pip install poetry`. Проверьте успешность установки командой `poetry --version`
- В терминале откройте папку с файлами проекта
- Для установки игры введите команду `make install`
- Для запуска игры введите команду `make project`
- Если не знаете, с чего начать, введите команду `help`
